DOM Manipulation Techniques

  • querySelector -> (everything)select by .class, #id, (h1 or body)element
  • getElementsByClassName "classname" .class -> select by class
  • getElementById "id" #id -> select by id (every id must be unique on a single webpage)
  • getElementsByTagName "li or ul"
  • (anything after the TagName is attribute) like "class" is a attribute.
